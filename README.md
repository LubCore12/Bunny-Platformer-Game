# 🐇 Rabbit With a Gun

A fast-paced **pixel-art survival platformer** made with **Python 3.12** and **Pygame** 🎮

In this game, you play as a **rabbit with a gun** and try to survive for as long as possible. Your enemies are **worms** and **bees**, and your goal is simple: **fight, dodge, and stay alive** for the maximum amount of time.

## ✨ Features

- Pixel-art platformer gameplay
- Play as a rabbit armed with a pistol 🔫
- Survival-focused combat
- Enemies include worms 🪱 and bees 🐝
- Smooth and enjoyable animations
- Simple and responsive controls
- Built with Python 3.12 and Pygame

## 🎮 Controls

| Action | Key / Button |
|---|---|
| Move left | `A` |
| Move right | `D` |
| Jump | `Space` |
| Shoot | `Left Mouse Button` |

## 📦 Installation

The steps below use **venv** and **requirements.txt** and work on **Windows, Linux, and macOS**.

### 1. Clone the repository

```bash
git clone <your-repository-url>
cd <your-project-folder>
```

### 2. Create a virtual environment

```bash
python -m venv venv
```

If `python` does not work on your system, try:

```bash
python3 -m venv venv
```

### 3. Activate the virtual environment

#### Windows (Command Prompt)

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the game

```bash
python code/main.py
```

If needed, use:

```bash
python3 code/main.py
```

## ✅ Requirements

Make sure you have:

- **Python 3.12** installed

## 📝 Notes

- It is recommended to run the game inside a virtual environment.
